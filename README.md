# Headhunter AI Skills

Public preview of a recruiting delivery skill suite created by Declan Zhao / Zhao Advisory.

> This repository is a public preview, not an open-source project and not an installable skill distribution.

## What This Is

Headhunter AI Skills is a set of AI-assisted recruiting delivery workflows for executive search consultants and recruiting teams.

The suite is designed to help consultants move from scattered notes and one-off prompts into a repeatable delivery workflow:

- understand clients and public signals
- calibrate roles before search
- judge candidate communication strategy
- generate resume recommendation reports
- prepare candidates for interviews
- debrief interviews and decide next actions
- design offer-stage strategy
- track follow-up nodes and process risk

## Public Preview Scope

This public repository shows:

- the workflow map
- each skill's public positioning
- typical inputs and outputs
- sample fragments based on fictional scenarios
- boundaries, privacy rules, and licensing path

This public repository does not include:

- the complete internal `SKILL.md` files

## 中文能力说明

1. **客户分析 Skill**：用于新客户开发、候选人 brief、月度客户更新和 BD 前预判。输出客户画像、人才吸引点、市场含义、合作风险和顾问追问。
2. **舆情线索 Skill**：用于客户口碑复杂、候选人有顾虑或岗位受公司声誉影响的场景。输出来源、日期、原始说法、证据等级和待核实缺口。
3. **岗位分析 Skill**：用于 JD 模糊、标准摇摆或客户需求前后不一致的岗位。输出真实任务、必须项、妥协项、矛盾点和 intake 问题。
4. **沟通话术 Skill**：用于候选人匹配度不确定、首次触达、跟进或冷场恢复。先判断 level、适配度和风险，再生成话术。
5. **简历报告 Skill**：用于候选人正式推荐前，把简历、沟通记录和岗位要求整理成客户能读、顾问能控的推荐报告。
6. **面试辅导 Skill**：用于 HR 面、高管面、直属上级面和跨部门协同面。输出顾问辅导重点、电话沟通稿、候选人可读笔记和风险提醒。
7. **整体评估 Skill**：用于面试后反馈不完整、双方信号冲突或 Offer 前信息不清楚的流程。输出证据完整度、双向意向、错配诊断和下一步建议。
8. **Offer 谈判 Skill**：用于薪酬、title、汇报线、入职时间、竞 offer 或反悔风险影响成交的阶段。输出 pre-close 检查、结构设计、BATNA 和双方话术。
9. **流程跟进 Skill**：用于面试、Offer、离职、入职、保证期等节点的持续推进。输出流程快照、提醒节点、关键问题、风险信号和跟进记录。
